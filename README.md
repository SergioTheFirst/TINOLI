# ✨ Magic Stitch — Portfolio Website

> Сайт-портфолио мастера украшений ручной работы.  
> Тёмный премиум-дизайн. Без баз данных. Без хостинга. Полная переносимость.

---

## 🚀 Быстрый старт

```bash
python server.py
```

Откроется браузер на `http://localhost:8000` — сайт готов.

---

## 📁 Структура проекта

```
magicstitch/
├── index.html          # Главная страница
├── gallery.html        # Галерея работ
├── about.html          # О мастере
├── contact.html        # Контакты
├── admin.html          # Форма загрузки (только локально!)
├── server.py           # Локальный Python сервер
├── config.json         # ВСЕ тексты сайта и контакты
├── css/
│   └── style.css
├── js/
│   └── main.js
└── works/              ← ВЕСЬ КОНТЕНТ ЗДЕСЬ
    ├── works.json      # Список всех работ
    └── images/         # Фотографии украшений
```

---

## ➕ Как добавить новую работу

1. Запусти `python server.py`
2. Открой `http://localhost:8000/admin.html`
3. Выбери фото → напиши описание → нажми **«Опубликовать»**
4. Работа сразу появляется на сайте

---

## ✏️ Как изменить тексты и контакты

Открой `config.json` в любом текстовом редакторе (Блокнот, VS Code).

```json
{
  "site": {
    "name": "Magic Stitch",
    "tagline": "Эксклюзивные украшения ручной работы"
  },
  "contact": {
    "phone": "+7 (999) 123-45-67",
    "instagram": "@magicstitch"
  }
}
```

Сохранил → обновил страницу → изменения применились.

## 📦 Переносимость

| Что перенести | Как |
|--------------|-----|
| Все работы | Скопируй папку `works/` |
| Все тексты | Скопируй `config.json` |
| Весь сайт | Скопируй всю папку проекта |

На новом хостинге — просто скопируй файлы. Никаких баз данных, никаких миграций.

---

## 🛠 Требования

- Python 3.10+
- Windows 10 / macOS / Linux
- Браузер (Chrome, Firefox, Edge)

Никаких дополнительных библиотек — только стандартная библиотека Python.

---

## 🌐 Деплой на хостинг

Для публикации в интернете скопируй все файлы **кроме** `server.py` и `admin.html` на любой статический хостинг:

- [GitHub Pages](https://pages.github.com/) — бесплатно
- [Netlify](https://netlify.com/) — бесплатно
- Любой обычный хостинг (папка `public_html`)

`admin.html` и `server.py` используются **только локально** на твоём компьютере.
