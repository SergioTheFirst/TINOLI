# TINOLI — There Is No Other Like It

> Премиальное портфолио мастера авторских украшений ручной работы.  
> Тёмный дизайн. Золотые акценты. Чистый HTML/CSS/JS. Полная переносимость.

**Сайт:** [tinoli.ru](https://tinoli.ru)  
**Деплой:** Cloudflare Pages (автоматически при пуше в GitHub)

---

## Быстрый старт

```bash
python server.py
```

Сайт доступен на `http://localhost:8000`.  
Админ-панель — `http://localhost:8000/admin.html`.

---

## Структура проекта

```
TINOLI/
├── index.html          # Главная: герой, избранные работы, превью «О мастере»
├── gallery.html        # Галерея с фильтрацией по категориям
├── about.html          # О мастере

├── admin.html          # Админ-панель (только локально)

├── config.json         # Все тексты, контакты, SEO-метаданные
├── favicon.svg         # Логотип-иконка
├── CNAME               # Домен tinoli.ru
├── css/
│   └── style.css       # Стили: тёмно-фиолетовая палитра, золотые акценты
├── js/
│   └── main.js         # Галерея, лайтбокс, анимации, загрузка конфига
└── works/
    ├── works.json      # Каталог работ (название, категория, описание, файлы)

```

## Как добавить работу

1. Запусти `python server.py`
2. Открой `http://localhost:8000/admin.html`
3. Вкладка **«Добавить работу»** → выбери фото, заполни поля → **«Опубликовать»**
4. Работа мгновенно появляется на сайте

---

## Как изменить тексты и контакты

Отредактируй `config.json`:

```json
{
  "site": {
    "name": "TINOLI",
    "tagline": "Авторские украшения ручной работы"
  },
  "contact": {
    "telegram": "@tinoli",
    "instagram": "@tinoli",
    "email": "hello@tinoli.ru"
  }
}
```

Сохранил → обновил страницу → готово.  
Или используй вкладку **«Настройки»** в админ-панели.

---

## Публикация на GitHub

1. Создай [Personal Access Token](https://github.com/settings/tokens) с правом `repo`
2. Админ-панель → **«Публикация»** → вставь токен → **«Сохранить токен»**
3. Нажми **«Опубликовать на GitHub»** — все файлы уходят в репозиторий
4. Cloudflare Pages подхватывает изменения автоматически (~30 сек)

Токен хранится в `secrets.json` (не коммитится, в `.gitignore`).

## Переносимость

| Что | Как |
|-----|-----|
| Все работы | Скопируй `works/` |
| Все тексты | Скопируй `config.json` |
| Весь сайт | Скопируй всю папку проекта |

Никаких баз данных, npm, фреймворков. Только файлы.
