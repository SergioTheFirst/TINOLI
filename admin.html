Публикация на GitHub

Отправляет все файлы сайта на GitHub через API.
Git не нужен — работает на любом компьютере с Python.

Как получить токен (один раз)

  1. Откройте github.com/settings/tokens
  2. Generate New Token (classic) → галочка repo → Generate
  3. Скопируйте токен и вставьте в поле выше
